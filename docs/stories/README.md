## Visual Testing Strategy

Each story includes a **Visual Testing Checklist** for validating UI implementation:

1. **Screenshots** at key breakpoints (375px, 768px, 1280px)
2. **Video recordings** for animations
3. **Browser DevTools** for color contrast verification
4. **Lighthouse/aXe** for accessibility audits
5. **Real device testing** on iOS Safari and Android Chrome

### Recommended Testing Workflow

1. Run local dev server: `cd app && bun run dev`
2. Open browser at http://localhost:5173
3. Use Chrome DevTools device toolbar for responsive testing
4. Take screenshots at each breakpoint
5. Record animations using QuickTime or browser recorder
6. Run Lighthouse audit in DevTools
7. Test with VoiceOver (Cmd+F5 on Mac)
